# Platform Icons Guide

This folder contains the platform logo icons used in the chat display.

## Required Icons

Place the following logo files in the `icons/` subfolder:

### Icon Files Needed:
- `twitch.png` - Twitch logo (recommended: 24x24px or 32x32px)
- `youtube.png` - YouTube logo
- `kick.png` - Kick logo
- `trovo.png` - Trovo logo
- `dlive.png` - DLive logo
- `twitter.png` - Twitter/X logo

## Where to Get Official Logos

### Twitch
- Official Brand Assets: https://brand.twitch.tv/
- Download the Glitch (icon) logo
- Use PNG format with transparent background
- Recommended: Purple glitch icon on transparent background

### YouTube
- Official Brand Resources: https://www.youtube.com/howyoutubeworks/resources/brand-resources/
- Download the YouTube icon (red play button)
- Use PNG format

### Kick
- Official website: https://kick.com/
- Right-click on their logo and save
- Or screenshot and crop the logo

### Trovo
- Official website: https://trovo.live/
- Brand resources may be available in their press kit
- Save logo as PNG

### DLive
- Official website: https://dlive.tv/
- Save logo from their site

### Twitter/X
- Official Brand Toolkit: https://about.twitter.com/en/who-we-are/brand-toolkit
- Download the bird logo or new X logo
- Use PNG format

## Icon Specifications

- **Format**: PNG with transparent background (preferred)
- **Size**: 24x24px to 32x32px recommended
- **Color**: Use official brand colors when possible
- **Background**: Transparent or match app theme (#1e1e1e)
